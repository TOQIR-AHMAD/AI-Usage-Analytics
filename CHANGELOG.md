# Changelog

## 1.0.1

- Added an extension icon (purple gradient + pulse line).

## 1.0.0

- Initial release.
- Status bar items per configured AI tool (Claude, Copilot, Codex, Cursor, Gemini, CodeWhisperer, Tabnine).
- Sidebar dashboard webview with overview cards, per-tool rows, 7-day charts, and session history.
- Notification alerts at configurable thresholds (default 70%, 90%, 100%).
- Optional daily digest notification.
- API keys stored in `vscode.SecretStorage`. No telemetry.
- CSV export of daily usage.
